## 3. Run Locally

This repository is static documentation and does not require a build step.

```bash
git clone https://github.com/GEEKProtocol0110/geek-protocol-docs.git
cd geek-protocol-docs
python3 -m http.server 8000
```

Open `http://localhost:8000` in your browser.

## 4. Contribute Changes

```bash
git checkout -b docs/your-change-name
# edit files
git add .
git commit -m "docs: describe your update"
git push origin docs/your-change-name
```

Then open a pull request against `main`.
